# HANDOFF

## Current Status

**Numera** is a SwiftUI iOS 17+ expense tracker backed by Supabase
(auth + Postgres with RLS). The Xcode project is generated by XcodeGen
(`project.yml`); CI compile-checks pushes and Fastlane/Match deploys to TestFlight.

The app has five tabs on the native (Liquid Glass on iOS 26) tab bar — Home,
Activity, Insights, Budget, Settings — plus a floating (+) to add transactions
from anywhere. Typography is SF Pro Rounded app-wide; charts are native Swift
Charts.
Features and card anatomy follow the Quanto reference app (screenshots in
`quanto-app/`) while keeping Numera's dark/mint design language
(`luminous_ledger/DESIGN.md`).

Data model (schema v2): user-editable categories (emoji/color/kind/order),
accounts (emoji + starting balance), transactions (category_id/account_id FKs),
budgets (overall + per-category). See `supabase/README.md`.

Monetization: **Numera Pro** (StoreKit 2 — monthly / yearly with 14-day trial /
lifetime). Budgeting, recurring transactions, and CSV export are Pro; entitlements
are tracked on-device by `PremiumManager`.

## Session — Rounded type, native Swift Charts, Insights ranges (2026-07-06, latest)

Branch: `feature/liquid-glass-ui`, merged into `main` at the start of the session
(user asked to push the last update first). Changelog version: **0.14.0**.

**What changed**
- **Type:** SF Pro Rounded everywhere. `AppTypography` dropped the
  `PlusJakartaSans` customs (never registered — they silently fell back to plain
  SF, which is the font the user disliked); every `.system(size:)` now passes
  `design: .rounded` (scripted sweep); root sets `.fontDesign(.rounded)`;
  `UINavigationBar` large/inline title fonts set to rounded in `NumeraApp.init`
  (fonts only — no background overrides, so iOS 26 bar glass is intact).
- **Charts → native Swift Charts** (`import Charts`), same public APIs:
  - `DayBarsChart`: `BarMark` + dashed `RuleMark` average; the average value is a
    trailing annotation **without** the ⓘ badge (per `error-images/average-line-*.png`);
    tapping it opens the new Quanto-style `AverageInfoSheet` ("Got it" pill).
    Y-axis: `0` + niced max on the right. Day labels can't wrap anymore.
  - `MonthlyBarsChart`: grouped `BarMark` pairs, `.chartXSelection` for bar taps
    (transient value mapped to `onSelect`; selection stays in the caller),
    selected axis label gets the Quanto capsule, `chartXScale(domain:)` keeps
    data order (Charts would sort the month names alphabetically otherwise).
  - `DonutChart`: `SectorMark` (`innerRadius: .inset(lineWidth)`, rounded caps,
    angular inset), `.chartAngleSelection` mapped back to a segment index, track
    ring drawn in `chartBackground` using `proxy.plotFrame`.
- **Insights:** native large title + segmented **Weekly / Monthly / Quarterly /
  Yearly** picker at the top. New `PeriodUnit` in `Services/Period.swift` with
  `period(of:containing:)` / `shift(by:unit:)` / `title` / `shortLabel`;
  `PeriodMath.days` cap raised 62 → 400 so yearly `dailyTotals` work. Month
  picker + calendar only in Monthly; series = 6 weeks/months/quarters, 5 years;
  change badge inlines the previous-period comparison (unit-aware). The donut
  hole absorbs taps (and clears the segment selection) so the center month
  button area can't select slices.
- **Calendar:** glass day cells (one `LiquidGlassGroup`; today = accent-tinted
  interactive glass; fallback = flat fills, not 31 materials); container card is
  now solid (no glass-on-glass); fixed 46pt cells + `minimumScaleFactor(0.5)`
  amounts fix the overflow the user screenshotted.
- **Budget:** pencils removed (overall header button + grid badges). Whole
  overall card is a Button with a context menu. `BudgetEditSheet` now shows the
  tapped card at the top — a live `ringPreview` recomputed from the amount field
  as you type — then amount / Save / Remove; auto-focuses the field for new
  budgets. Native large title.
- **Activity:** the list shows the whole history (grouped by day, `LazyVStack`);
  the selected month only scopes the hero total + chart (`periodFiltered` vs
  `filtered`); non-current-year headers include the year. `SelectMonthSheet`
  years always reach 2020. Settings also moved to a native large title;
  `PageTitle.swift` deleted (unused).
- **(+) lag fix:** sheet state isolated in `AddTransactionFAB` (ContentView) —
  tapping no longer re-renders the entire `TabView` (Insights was recomputing
  every card synchronously before the sheet appeared).
- **What's New:** SF-symbol tiles (sparkles / chart.bar.xaxis / calendar /
  clock.arrow.circlepath / arrow.triangle.2.circlepath) and 0.14 copy.

**Files touched:** `AppTypography`, `NumeraApp`, `ContentView`, `Period.swift`,
`DayBarsChart`, `MonthlyBarsChart`, `DonutChart`, `CalendarSpendGrid`,
